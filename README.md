# RAG Profile Matching

AI-powered Resume Matching System using Retrieval-Augmented Generation (RAG), ChromaDB, and Semantic Search.

## Features
- Resume parsing
- Embedding generation
- Vector database search
- Job description matching
- Candidate ranking

## Tech Stack
- Python
- ChromaDB
- Sentence Transformers

```bash
python3 resume_rag.py
